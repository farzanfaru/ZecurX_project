Web Vulnerability Scanner - Setup Guide

1. Requirements:
- Python 3 installed
- Internet connection

2. Install Required Libraries:
pip install requests

3. Tool Location:
Navigate to the tool directory:
cd tool

4. Run the Tool:
python3 advanced_scanner.py

5. Input Format:
Enter target URL with parameters for testing.

Example:
http://localhost/DVWA/vulnerabilities/xss_r/?name=
http://testphp.vulnweb.com/listproducts.php?cat=

6. Features:
- Directory Scanning
- XSS Payload Testing
- SQL Injection Testing

7. Output:
- Displays detected vulnerabilities
- Shows successful payloads
- Provides summary report

8. Limitations:
- Does not support login/session handling
- Limited support for JavaScript-based applications
- Works best with simple parameter-based URLs

9. Usage Note:
This tool is designed for educational purposes and should only be used in authorized environments.
